title: Supported AI Platforms
description: All AI platforms supported by ThairaAI.
---

ThairaAI supports 20+ AI platforms. You can switch between them at any time from Settings → Models.

## Official platforms

| Platform    | Models                          | Notes                  |
| ----------- | ------------------------------- | ---------------------- |
| Anthropic   | Claude 3.5 Sonnet, Haiku, Opus  | API key required       |
| Google      | Gemini 2.0, 1.5 Pro/Flash       | Free with Google login |
| OpenAI      | GPT-4o, o1, o3                  | API key required       |

## Cloud providers

| Platform    | Notes                                    |
| ----------- | ---------------------------------------- |
| AWS Bedrock | Access via Thaira CLI (aionrs backend)   |
| DeepSeek    | API key required                         |
| OpenRouter  | Unified gateway to many models           |
| New API     | Self-hosted unified model gateway        |

## Local models

| Platform   | Notes                                          |
| ---------- | ---------------------------------------------- |
| Ollama     | Run any model locally via `ollama serve`       |
| LM Studio  | Configure via Custom platform + local endpoint |

## Other platforms

Moonshot (Kimi), Qwen (Dashscope), Zhipu, Baidu Qianfan, Tencent Hunyuan, MiniMax, xAI, SiliconFlow, Volcengine Ark, Poe, StepFun.

## Free option

Sign in with Google to use Gemini for free — no API key needed. The Gemini free tier is generous enough for most day-to-day tasks.
