title: Google Calendar
description: Using ThairaAI with Google Calendar.
---

The AI agent can read your calendar events and help you plan, schedule, and prepare for meetings.

## What you can ask

```
What's on my calendar this week?
```
```
Do I have any conflicts on Thursday afternoon?
```
```
Summarize tomorrow's meetings
```
```
Create an event for a team sync next Monday at 10am
```

## Permissions

| Scope                  | Purpose                          |
| ---------------------- | -------------------------------- |
| `calendar.readonly`    | Read events and availability     |
| `calendar.events`      | Create and update events         |

You can revoke access at any time from [Google Account Permissions](https://myaccount.google.com/permissions).

## Privacy

Calendar data is fetched on demand. Nothing is stored remotely.
