title: Google Agenda
description: Usando o ThairaAI com o Google Agenda.
---

O agente de IA pode ler seus eventos de calendário e ajudá-lo a planejar, agendar e se preparar para reuniões.

## O que você pode pedir

```
O que tenho na agenda essa semana?
```
```
Tenho algum conflito na quinta à tarde?
```
```
Resuma as reuniões de amanhã
```
```
Crie um evento para uma sincronização de equipe na próxima segunda às 10h
```

## Permissões

| Escopo                 | Finalidade                           |
| ---------------------- | ------------------------------------ |
| `calendar.readonly`    | Ler eventos e disponibilidade        |
| `calendar.events`      | Criar e atualizar eventos            |

Você pode revogar o acesso a qualquer momento em [Permissões da Conta Google](https://myaccount.google.com/permissions).

## Privacidade

Os dados da agenda são buscados sob demanda. Nada é armazenado remotamente.
