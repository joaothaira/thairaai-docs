---
title: Painel de Visualização
description: Veja arquivos gerados pela IA sem sair do ThairaAI.
---

O painel de visualização abre automaticamente quando o agente gera ou modifica um arquivo. Sem precisar trocar de app.

## Formatos suportados

| Categoria  | Formatos                                                                   |
| ---------- | -------------------------------------------------------------------------- |
| Documentos | PDF, Word (`.doc`, `.docx`), Excel (`.xls`, `.xlsx`, `.csv`), PowerPoint (`.ppt`, `.pptx`) |
| Código     | JS, TS, Python, Go, Rust, Java, C/C++, CSS, JSON, YAML, Shell e 30+ mais  |
| Marcação   | Markdown, HTML                                                             |
| Imagens    | PNG, JPG, GIF, SVG, WebP, BMP, AVIF                                        |
| Outros     | Arquivos diff/patch                                                        |

## Funcionalidades

- **Atualização ao vivo** — atualiza automaticamente conforme o agente edita o arquivo
- **Editável** — arquivos Markdown, código e HTML podem ser editados diretamente no painel
- **Multi-abas** — abra vários arquivos simultaneamente
- **Histórico de versões** — visualize e restaure versões anteriores (baseado em Git)

## Abrindo o painel

O painel abre automaticamente após a geração de um arquivo. Você também pode abri-lo manualmente clicando em qualquer referência de arquivo no chat.
