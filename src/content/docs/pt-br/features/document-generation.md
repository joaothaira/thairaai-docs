title: Geração de Documentos
description: Gere PPT, Word e Excel a partir de linguagem natural.
---

O ThairaAI pode gerar documentos de escritório prontos para uso por meio de suas habilidades integradas.

## PPT (PowerPoint)

```
Crie uma apresentação de 10 slides para um pitch de uma ferramenta SaaS de produtividade para equipes remotas
```

Saída: `.pptx` editável com transições animadas Morph.

O assistente **Morph PPT** gera apresentações com transições suaves entre slides usando o PowerPoint Morph. Troque para ele pelo seletor de assistentes.

## Documentos Word

```
Escreva uma proposta técnica para migrar nossa infraestrutura para Kubernetes
```

Saída: `.docx` formatado pronto para compartilhar ou editar no Word.

O assistente **Criador de Word** é otimizado para escrita longa e estruturada — relatórios, propostas, artigos acadêmicos, contratos.

## Planilhas Excel

```
Crie um modelo financeiro para uma projeção de startup de 3 anos com receita, despesas e taxa de consumo
```

Saída: `.xlsx` com fórmulas, gráficos e formatação profissional.

O assistente **Criador de Excel** gera planilhas com análise, gráficos e estilização automática.

## Painel de visualização

Os arquivos gerados abrem imediatamente no painel de visualização integrado — sem precisar abrir outro app. A visualização atualiza ao vivo enquanto o agente edita o arquivo.

Formatos suportados: PDF, Word, Excel, PowerPoint, Markdown, HTML, arquivos de código, imagens, diffs.

Veja [Painel de Visualização](/pt-br/features/preview-panel/) para detalhes.
