---
title: Agente Integrado
description: O agente de IA integrado que acompanha o ThairaAI.
---

O ThairaAI já vem com um motor de agente de IA completo. Você não precisa instalar Claude Code, Gemini CLI ou qualquer outra ferramenta CLI para começar.

## Funcionalidades

| Funcionalidade        | Detalhes                                                       |
| --------------------- | -------------------------------------------------------------- |
| Leitura/escrita de arquivos | Ler, criar, editar e deletar arquivos na sua máquina   |
| Busca na web          | Pesquisar na web e recuperar conteúdo de páginas               |
| Geração de imagens    | Gerar imagens via Gemini                                       |
| Ferramentas MCP       | Usar qualquer servidor Model Context Protocol                  |
| Execução de código    | Rodar scripts e comandos                                       |
| Saída de documentos   | Gerar PPT, Word e Excel via habilidades integradas             |

## Como usar

Basta digitar no chat. O agente decide quais ferramentas usar com base na sua mensagem.

```
Leia o arquivo CSV na minha pasta Downloads e me dê um resumo
```
```
Pesquise na web os preços mais recentes da API do Claude
```
```
Crie um documento Word com uma proposta de projeto para um app mobile
```

## Assistentes

O agente pode operar em diferentes modos via **Assistentes** — personas pré-configuradas otimizadas para tarefas específicas:

- **Cowork** — execução autônoma de tarefas gerais
- **Criador de PPT** — geração de apresentações com animações Morph
- **Criador de Word** — escrita de documentos longos e estruturados
- **Criador de Excel** — geração e análise de planilhas
- **Escritor Acadêmico** — estrutura e escrita de artigos de pesquisa
- **UI/UX Pro Max** — design de interfaces em 57 estilos

Troque de assistente pelo seletor no cabeçalho do chat.

## Ferramentas MCP

O ThairaAI suporta Model Context Protocol. Adicione qualquer servidor MCP em **Configurações → Ferramentas → MCP** e ele fica disponível para o agente automaticamente. Uma configuração se aplica a todos os agentes.
