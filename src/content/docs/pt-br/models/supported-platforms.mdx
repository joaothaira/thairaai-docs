O ThairaAI suporta mais de 20 plataformas de IA. Você pode trocar entre elas a qualquer momento em Configurações → Modelos.

## Plataformas oficiais

| Plataforma  | Modelos                          | Observações               |
| ----------- | -------------------------------- | ------------------------- |
| Anthropic   | Claude 3.5 Sonnet, Haiku, Opus   | Chave de API necessária   |
| Google      | Gemini 2.0, 1.5 Pro/Flash        | Gratuito com login Google |
| OpenAI      | GPT-4o, o1, o3                   | Chave de API necessária   |

## Provedores cloud

| Plataforma  | Observações                                      |
| ----------- | ------------------------------------------------ |
| AWS Bedrock | Acesso via Thaira CLI (backend aionrs)           |
| DeepSeek    | Chave de API necessária                          |
| OpenRouter  | Gateway unificado para vários modelos            |
| New API     | Gateway de modelos auto-hospedado                |

## Modelos locais

| Plataforma  | Observações                                              |
| ----------- | -------------------------------------------------------- |
| Ollama      | Rode qualquer modelo localmente via `ollama serve`       |
| LM Studio   | Configure via plataforma Custom + endpoint local         |

## Outras plataformas

Moonshot (Kimi), Qwen (Dashscope), Zhipu, Baidu Qianfan, Tencent Hunyuan, MiniMax, xAI, SiliconFlow, Volcengine Ark, Poe, StepFun.

## Opção gratuita

Entre com o Google para usar o Gemini gratuitamente — sem precisar de chave de API. O nível gratuito do Gemini é generoso o suficiente para a maioria das tarefas do dia a dia.
