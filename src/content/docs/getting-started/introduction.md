ThairaAI is a free, open-source desktop app that puts an AI agent on your computer. It is built on top of [AionUi](https://github.com/iOfficeAI/AionUi) and adds Google Calendar and Gmail integration.

## What makes it different

Most AI tools give you a chat box. ThairaAI gives you an agent that can actually work on your machine — reading and writing files, browsing the web, generating documents, and running scheduled tasks.

You see everything the agent does. You stay in control.

## Key capabilities

- **Built-in agent** — works the moment you install it, no CLI tools needed
- **Google integration** — connect Gmail and Calendar with a single sign-in
- **Multi-agent** — use Claude Code, Codex, Gemini CLI and others through one interface
- **WebUI** — access from any browser or phone on your network
- **Scheduled tasks** — cron-based automation that runs while you sleep
- **Office documents** — generate PPT, Word, and Excel files from natural language
- **20+ AI platforms** — Anthropic, Gemini, OpenAI, DeepSeek, Ollama, and more

## Tech stack

ThairaAI is an Electron app built with:

- [Electron](https://www.electronjs.org/) — cross-platform desktop runtime
- [Vite](https://vitejs.dev/) — build tooling
- [React](https://react.dev/) — UI
- [Bun](https://bun.sh/) — package manager and runtime
