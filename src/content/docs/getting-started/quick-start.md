<Steps>

1. **Install ThairaAI**

   Download from [GitHub Releases](https://github.com/joaothaira/ThairaAI/releases) and install for your platform.

2. **Sign in**

   Open ThairaAI and sign in with your Google account — this gives you free access to Gemini and connects Gmail and Calendar automatically.

   Alternatively, paste any API key (Anthropic, OpenAI, DeepSeek, etc.) in Settings → Models.

3. **Start a conversation**

   Type a message in the chat. The built-in agent is ready to go — no extra setup needed.

   Try something like:
   - *"Summarize my emails from today"*
   - *"What's on my calendar this week?"*
   - *"Create a project plan in a Word document"*
   - *"Organize the files in my Downloads folder"*

4. **Connect Google (optional)**

   If you signed in with Google, your Gmail and Calendar are already connected.
   Go to **Settings → Integrations** to review permissions.

5. **Add more AI agents (optional)**

   If you have Claude Code, Codex, or Gemini CLI installed, ThairaAI auto-detects them.
   Go to **Settings → Agents** to manage which agents are active.

</Steps>
