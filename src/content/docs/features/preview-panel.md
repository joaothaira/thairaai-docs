---
title: Preview Panel
description: View AI-generated files without leaving ThairaAI.
---

The preview panel opens automatically when the agent generates or modifies a file. No need to switch apps.

## Supported formats

| Category   | Formats                                                   |
| ---------- | --------------------------------------------------------- |
| Documents  | PDF, Word (`.doc`, `.docx`), Excel (`.xls`, `.xlsx`, `.csv`), PowerPoint (`.ppt`, `.pptx`) |
| Code       | JS, TS, Python, Go, Rust, Java, C/C++, CSS, JSON, YAML, Shell, and 30+ more |
| Markup     | Markdown, HTML                                            |
| Images     | PNG, JPG, GIF, SVG, WebP, BMP, AVIF                       |
| Other      | Diff/patch files                                          |

## Features

- **Live tracking** — automatically updates as the agent edits the file
- **Editable** — Markdown, code, and HTML files can be edited directly in the panel
- **Multi-tab** — open multiple files simultaneously
- **Version history** — view and restore previous versions (Git-based)

## Opening the panel

The panel opens automatically after file generation. You can also open it manually by clicking any file reference in the chat.
