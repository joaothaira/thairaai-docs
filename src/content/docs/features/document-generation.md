ThairaAI can generate production-ready office documents through its built-in skills.

## PPT (PowerPoint)

```
Create a 10-slide pitch deck for a SaaS productivity tool targeting remote teams
```

Output: editable `.pptx` with Morph-animated slide transitions.

The **Morph PPT** assistant generates presentations with smooth slide-to-slide transitions powered by PowerPoint Morph. Switch to it from the assistant selector.

## Word documents

```
Write a technical proposal for migrating our infrastructure to Kubernetes
```

Output: formatted `.docx` ready to share or edit in Word.

The **Word Creator** assistant is optimized for structured long-form writing — reports, proposals, research papers, contracts.

## Excel spreadsheets

```
Create a financial model for a 3-year startup projection with revenue, expenses, and burn rate
```

Output: `.xlsx` with formulas, charts, and auto-formatting.

The **Excel Creator** assistant generates spreadsheets with analysis, charts, and professional styling.

## Preview panel

Generated files open immediately in the built-in preview panel — no need to open another app. The preview updates live as the agent edits the file.

Supported formats: PDF, Word, Excel, PowerPoint, Markdown, HTML, code files, images, diffs.

See [Preview Panel](/features/preview-panel/) for details.
